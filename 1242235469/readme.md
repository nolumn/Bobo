# bobo

a roblox script for ball & axe

### usage
main version
```lua
loadstring(game:HttpGet('https://github.com/nolumn/Bobo/raw/refs/heads/main/bobaloader.lua'))('main')
```
lite version (for legit purposes)
```lua
loadstring(game:HttpGet('https://github.com/nolumn/Bobo/raw/refs/heads/main/bobaloader.lua'))('lite')
```

## pogo

> press **space** to launch your ball toward your cursor. shows a trajectory arc before you jump. touch supported.

![pogo screenshot](https://raw.githubusercontent.com/nolumn/Bobo/refs/heads/main/1242235469/screenshots/Screenshot%202026-03-20%20201559.png)
